# Desafio Decision Systems – Cadastro de Usuários
🚀 Para rodar localmente o projeto você precisa:

1. Ter o **[Docker](https://www.docker.com/)** instalado em sua máquina.

2. Clonar o repositório:
   ```
   git clone https://github.com/samuel-cardoso/desafio-decision-systems.git
   ```
   ```
   cd desafio-decision-systems
   ```

3. Rodar os containers com o Docker:
    ```
    docker compose up --build -d
    ```
  
  ## 🌐 Acessos que estarão disponíveis:
  
  Front End da aplicação: http://localhost:3000

  GraphQL Playground (Apollo Server): http://localhost:4000/graphql

  Prisma Studio (painel para ver os dados): http://localhost:5555
